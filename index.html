Vulnerabilities in TensorFlow

Overview of Identified CVEs

This section provides an overview of the CVEs related to vulnerabilities in TensorFlow. The data below is sourced from recent analyses of TensorFlow security issues.

CVE Number Description Severity Date
CVE-2021-29563 Memory corruption due to improper bounds checking in TensorFlow. High May 12, 2021
CVE-2021-29561 Buffer overflow when decoding JPEG images in TensorFlow. Medium April 5, 2021
CVE-2020-15200 Denial of Service (DoS) vulnerability when processing image formats. Low September 30, 2020

Technical Details

TensorFlow has been prone to several security vulnerabilities, ranging from buffer overflows to DoS attacks, especially in its image processing libraries. Proper patching and mitigation techniques should be employed to ensure safe usage.
